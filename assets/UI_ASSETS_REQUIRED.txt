SECTION 6: IMPLEMENTATION GUIDELINES
================================================================================

6.1 DESIGN PRINCIPLES
   - Color Scheme:
     * Dark Mode: Use dark grays (#1E1E1E, #2D2D2D, #3D3D3D), blue accent (#0078D4)
     * Light Mode: Use light grays (#F5F5F5, #EEEEEE, #E1E1E1), blue accent (#0078D4)
   
   - Consistency: All icons should follow same style (flat design recommended)
   
   - Sizing: Maintain consistent padding and proportions across all elements
   
   - Accessibility: Ensure adequate contrast ratios (WCAG AA minimum)

6.2 FILE NAMING CONVENTION
   - Use snake_case for all files
   - Include variant in filename: _dark, _light, _hover, _active
   - Example: button_download_dark_hover.png

6.3 TRANSPARENCY & OPTIMIZATION
   - Use PNG format with proper transparency for layering
   - Optimize all images for web (compress without quality loss)
   - Consider using SVG for scalability where applicable

6.4 INTEGRATION NOTES
   - Icons will be loaded in ui_enhanced.py and referenced in easycut.py
   - Implement fallback emoji if PNG assets fail to load
   - All assets should be optional (graceful degradation)
