# Static Website Files

This folder contains **static HTML files** for the EasyCut website and web services.

## Contents

### Public Website
- `index.html` - Main landing page
- `PRIVACY.html` - Privacy policy (HTML version)
- `TERMS.html` - Terms of service (HTML version)

### Verification Files
- `googlec68254c22a63edb3.html` - Google Site Verification file (required for OAuth)

## Usage
These files are typically served via GitHub Pages or deployed to a web server for:
- OAuth redirect URIs
- Privacy policy and terms display
- Google Search Console verification

## Deployment
When deploying the website, copy these files to the web server root or configure GitHub Pages to serve from this directory.
